# Typora 快捷键单页速查表（Windows）

> *最常用、最高频、**单页即可掌握全部核心操作**。*

---

## 📝 文本格式

| 功能        | 快捷键              |
| ----------- | ------------------- |
| 粗体        | Ctrl + B            |
| 斜体        | Ctrl + I            |
| 删除线      | Alt + Shift + 5     |
| 下划线      | Ctrl + U            |
| 行内代码    | `（反引号）         |
| 代码块      | Ctrl + Shift + `    |
| 撤销 / 重做 | Ctrl + Z / Ctrl + Y |

---

## 🔠 标题（Heading）

| 级别     | 快捷键     |
| -------- | ---------- |
| H1~H6    | Ctrl + 1~6 |
| 普通段落 | Ctrl + 0   |

---

## 📌 列表（Lists）
| 类型     | 快捷键           |
| -------- | ---------------- |
| 无序列表 | Ctrl + Shift + L |
| 有序列表 | Ctrl + Shift + O |
| 任务列表 | `[]` + 空格      |

---

## 📊 表格 / 引用 / 分割线

| 功能     | 快捷键 / 语法          |
| -------- | ---------------------- |
| 插入表格 | Ctrl + T               |
| 引用     | Ctrl + Shift + Q / `>` |
| 分割线   | `---` 或 `***` + 回车  |

---

## 🔗 链接与图片

| 功能     | 快捷键           |
| -------- | ---------------- |
| 插入链接 | Ctrl + K         |
| 插入图片 | Ctrl + Shift + I |

---

## 📄 文件操作
| 功能        | 快捷键              |
| ----------- | ------------------- |
| 打开 / 保存 | Ctrl + O / Ctrl + S |
| 关闭文件    | Ctrl + W            |
| 导出        | Ctrl + Shift + E    |
| 查找 / 替换 | Ctrl + F / Ctrl + H |
| 全屏        | F11                 |

---

## 🧱 块级元素
| 元素       | 触发方式                |
| ---------- | ----------------------- |
| 代码块     | ``` 或 Ctrl + Shift + ` |
| 数学公式块 | `$$` + 回车             |
| 列表       | `-` / `1.`              |

---

## ⭐ Typora 扩展

| 功能     | 语法       |
| -------- | ---------- |
| 目录 TOC | `[TOC]`    |
| 脚注     | `内容[^1]` |
