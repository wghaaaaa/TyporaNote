# 常用命令

----

## 1.快速生成标题

| 功能     | 快捷键                       |
| -------- | ---------------------------- |
| 一级标题 | # + 空格 后跟名称 + 回车     |
| 二级标题 | ## + 空格 后跟名称 + 回车    |
| 三级标题 | ### + 空格 后跟名称 + 回车   |
| 四级标题 | #### + 空格 后跟名称 + 回车  |
| 五级标题 | ##### + 空格 后跟名称 + 回车 |

## 2.生成代码块

| 功能       | 快捷键                         |
| ---------- | ------------------------------ |
| 生成代码块 | ~~~ + HTML（Java、CSS等）+回车 |

## 3.整行标红

| 功能     | 快捷键   |
| -------- | -------- |
| 整行标红 | > + 回车 |

## 4.字体加粗&标红

| 功能 | 快捷键                           |
| ---- | -------------------------------- |
| 加粗 | 字体左右添加**号，快捷键ctrl + B |
| 标红 | 字体左右添加``即可               |

## 5.序列

| 功能     | 快捷键       |
| -------- | ------------ |
| 无序列表 | - + 空格     |
| 有序列表 | 1 + . + 空格 |

## 6.表格

| 功能 | 快捷键   |
| ---- | -------- |
| 表格 | ctrl + T |

## 7.分割线

| 功能   | 快捷键     |
| ------ | ---------- |
| 分割线 | --- + 回车 |
